# Admin Dashboard – Responsive UI with Theme Switching

## Overview

This project is a responsive Admin Dashboard built using HTML, CSS Grid, and JavaScript.  
It demonstrates modern layout techniques, dynamic theme switching (dark/light mode), and persistent user preferences using localStorage.

The application highlights advanced front-end concepts such as CSS Grid template areas, CSS variables, and DOM-based class manipulation.

---

## Key Features

- Responsive grid-based layout using CSS Grid
- Dark / Light mode toggle
- Theme management using CSS variables
- Persistent theme preference using localStorage
- Clean dashboard-style UI
- Modular and scalable layout structure

---

## Functional Capabilities

- Structured dashboard sections (sidebar, header, main content)
- Dynamic theme switching
- Persistent theme across page reloads
- Responsive layout adaptation
- Organized grid template areas

---

## Technology Stack

- HTML5
- CSS3 (Grid + Variables)
- JavaScript (DOM manipulation & localStorage)


---

## Learning Outcomes

This project demonstrates:

- Advanced layout design using CSS Grid
- Theme management using CSS variables
- JavaScript-based UI state control
- LocalStorage for client-side persistence
- Responsive admin interface design

---

## How to Run

1. Clone the repository
2. Open `index.html` in a modern browser
3. Toggle between dark and light themes
4. Refresh the page to verify theme persistence
